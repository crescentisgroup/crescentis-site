# Crescentis Group - Site de Présentation

Bienvenue sur le dépôt officiel du site de présentation de **Crescentis Group™**.

## À propos

Crescentis Group est un écosystème transversal regroupant **4 pôles stratégiques**:

1. **Pôle IT & Digital** - Ingénierie informatique, développement web, solutions IA
2. **Pôle Communication & Événements** - Communication 360°, événementiel, audiovisuel
3. **Pôle AfriBuilding & Services** - Construction, nettoyage, entretien
4. **Pôle Consulting & Commerce** - Négoce, consulting, intermédiation commerciale

## Structure du Projet

```
├── index.html          # Page principale (HTML5)
├── style.css           # Feuille de styles responsive
├── script.js           # Logique JavaScript (menu, formulaire)
└── README.md           # Ce fichier
```

## Fonctionnalités

✅ **Design Responsive** - Compatible avec tous les appareils (mobile, tablette, desktop)  
✅ **Navigation Fluide** - Menu mobile et ancres de section  
✅ **Formulaire Intelligent** - Services dynamiques selon le pôle sélectionné  
✅ **Carrousel de Références** - Défilement continu des partenaires  
✅ **Charte Graphique Crescentis** - Couleurs : Marron (#543A20) et Bordeaux (#C61650)  

## Installation & Déploiement

### Localement

```bash
git clone https://github.com/crescentisgroup/crescentis-site.git
cd crescentis-site
# Ouvrir index.html dans un navigateur
```

### Via GitHub Pages

Ce site est automatiquement déployé sur : **https://crescentisgroup.github.io/crescentis-site**

## Personnalisation

- **Modifier les couleurs** : Éditer les variables CSS dans `style.css` (`:root`)
- **Ajouter du contenu** : Modifier les sections dans `index.html`
- **Intégrer une carte** : Remplacer la section `.map-placeholder` avec Google Maps
- **Connecter le formulaire** : Utiliser Formspree, EmailJS ou un backend custom
